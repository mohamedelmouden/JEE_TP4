Description :

Application Java JEE utilisant Hibernate ORM pour la persistance des entités Salle et Machine dans une base de données MySQL. Le projet suit le patron de conception DAO



Architecture du projet :


<img width="657" height="896" alt="image" src="https://github.com/user-attachments/assets/aca74502-8d56-4191-af0f-531345b4009f" />



Technologies utilisées : 

- java(JDK)21
- Hibernate ORM 5.6.9.Final
- MySQL Connector 8.0.29
- JPA API 2.2 
- JUnit 4.13.2
- Maven



Base de données :

Créer la base de données MySQL avant de lancer le projet : "base"




Lancer la classe principale :

src/main/java/test/Test.java → Run


Lancer les tests JUnit :

src/test/java/test/SalleServiceTest.java → Run
src/test/java/test/MachineServiceTest.java → Run
